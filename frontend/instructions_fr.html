Comment utiliser le service de Chatbot

Ce service vous permet de générer un plugin WordPress pour un chatbot IA personnalisé. Vous pouvez personnaliser le plugin en configurant votre base de connaissances, en définissant des questions/réponses prédéfinies et en sélectionnant des avatars personnalisés.

Étapes pour commencer

  1. Renseignez votre contenu :
    • Remplissez le champ « Base de connaissances » avec les détails de votre entreprise ou organisation.
    • Ajoutez une ou plusieurs paires Question-Réponse dans la zone prévue à cet effet.
  2. Sélectionnez ou téléchargez des avatars :
    • Choisissez un avatar parlant et un avatar inactif parmi les options prédéfinies ou téléchargez vos propres images personnalisées.
  3. Personnalisez l'apparence :
    • Définissez vos couleurs principale et secondaire pour le style du plugin.
  4. Email du propriétaire :
    • Saisissez votre adresse email pour recevoir des notifications des interactions du chatbot.
  5. Génération du plugin :
    • Cliquez sur « Soumettre » pour générer votre plugin personnalisé. Un fichier chatbot.zip sera créé et mis à disposition pour téléchargement.

Ajouter le plugin à votre site WordPress

  1. Connectez-vous à votre tableau de bord WordPress.
  2. Allez dans Extensions > Ajouter et cliquez sur le bouton Téléverser une extension.
  3. Sélectionnez le fichier chatbot.zip que vous avez téléchargé et cliquez sur Installer maintenant.
  4. Une fois l’extension installée, cliquez sur Activer l’extension.
  5. Pour afficher le chatbot, ajoutez le shortcode [chatbot_avatar language="fr-CA"] (ou [chatbot_avatar language="en-US"] pour l’anglais) dans n’importe quelle page ou article.

Exemple de flux de travail

Par exemple, si vous êtes dans l’immobilier, vous pourriez entrer dans votre base de connaissances : 

Vous êtes un assistant utile pour une entreprise immobilière appelée Mon Immobilier. 
Actuellement, nous opérons dans le centre-ville et disposons de plusieurs propriétés.

Et définir une paire FAQ telle que :

Question : Quelles sont vos heures d'ouverture ?
Réponse : Nous sommes ouverts de 9h à 17h, du lundi au vendredi.

Suivez les étapes ci-dessus pour générer votre plugin personnalisé et l’intégrer à votre site WordPress.
