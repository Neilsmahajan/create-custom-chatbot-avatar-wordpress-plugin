Comment utiliser le service de plugin d'avatar de chatbot

Ce service vous permet de générer un plugin WordPress d'avatar de chatbot AI personnalisé selon vos besoins. Vous pouvez personnaliser le plugin en configurant votre Base de Connaissances, vos Questions-Pré-définies et en sélectionnant des avatars personnalisés.

Prérequis : Récupération de vos clés API

  1. Clé API OpenAI :

    Rendez-vous sur la page des clés API OpenAI. Connectez-vous ou créez un compte, générez une nouvelle clé API et copiez la chaîne de caractères.

  2. Identifiants du compte de service Google Cloud Platform :

    Accédez à la Console Google Cloud et créez (ou sélectionnez) un projet.

    • Activez l’API Cloud Text-to-Speech et l’API Cloud Translation dans la bibliothèque d’API.
    • Allez dans IAM & Admin → Comptes de service et créez un nouveau compte de service en lui attribuant les autorisations requises.
    • Créez et téléchargez un fichier JSON pour ce compte de service.

    Vous téléchargerez ce fichier JSON lors de la génération de votre plugin.

Étapes pour démarrer

  1. Entrez votre contenu :
    • Remplissez le champ « Base de Connaissances » avec les détails de votre entreprise ou organisation.
    • Ajoutez une ou plusieurs paires Question-Réponse pré-définies dans l’espace prévu.
  2. Sélection ou téléchargement d’avatars :
    • Sélectionnez un avatar parlant et un avatar inactif parmi les options prédéfinies ou téléchargez vos propres images.
  3. Entrez vos clés API :
    • Entrez votre clé API OpenAI dans le champ prévu.
    • Téléchargez votre fichier JSON d’identifiants du compte de service GCP.
  4. Personnalisez l’apparence :
    • Choisissez vos couleurs Principale et Secondaire pour le style du plugin.
  5. Email du propriétaire :
    • Fournissez votre adresse email pour recevoir les interactions du chatbot.
  6. Génération du plugin :
    • Cliquez sur « Soumettre » pour générer votre plugin personnalisé. Un fichier chatbot.zip sera créé et disponible en téléchargement.

Ajouter le plugin à votre site WordPress

  1. Connectez-vous à votre tableau de bord WordPress (wp-admin).
  2. Naviguez vers Plugins > Ajouter puis cliquez sur le bouton Téléverser un plugin.
  3. Cliquez sur Choisir un fichier, sélectionnez le fichier chatbot.zip téléchargé, puis cliquez sur Ouvrir.
  4. Cliquez sur Installer maintenant.
  5. Une fois le plugin installé, cliquez sur Activer le plugin.
  6. Pour afficher le chatbot, ajoutez le shortcode [chatbot_avatar language="fr-CA"] (ou [chatbot_avatar language="en-US"] pour l’anglais) à une page ou un article.

Exemple de procédé

Par exemple, si vous exploitez une entreprise immobilière, vous pourriez entrer dans votre Base de Connaissances : 

Vous êtes un assistant fiable pour une entreprise immobilière nommée Mon Immobilier. Actuellement, nous opérons dans le centre-ville et disposons de plusieurs propriétés disponibles.

Et définir une paire FAQ comme :

Question : Quelles sont vos heures d'ouverture ?
Réponse : Nous sommes ouverts de 9 h à 17 h, du lundi au vendredi.

Suivez les étapes ci-dessus pour générer votre plugin personnalisé et l’intégrer à votre site WordPress.
